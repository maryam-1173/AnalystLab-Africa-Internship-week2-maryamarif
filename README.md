# Iris Dataset - Data Preprocessing & Exploratory Data Analysis (EDA)

## Project Overview

This project was completed as part of the AnalystLab Africa Machine Learning Internship (Week 2 Assignment).

The objective was to preprocess the Iris dataset, perform Exploratory Data Analysis (EDA), and prepare the data for future machine learning model development.

##  Dataset

- Dataset: Iris Dataset
- Target Variable: Species
- Total Records: 150
- Features: Sepal Length, Sepal Width, Petal Length, Petal Width, Species

##  Tasks Performed

- Data Loading & Inspection
- Missing Value Analysis
- Duplicate Record Check
- Outlier Detection
- Feature Selection
- Label Encoding
- Feature Scaling using StandardScaler
- Exploratory Data Analysis (EDA)
- Correlation Matrix & Heatmap
- Machine Learning Readiness Summary

## Visualizations

- Histograms
- Boxplots
- Count Plot
- Scatter Plots
- Correlation Heatmap

##  Key Insights

- No missing values or duplicate records were found.
- Petal Length and Petal Width are the most important features for species classification.
- The dataset is perfectly balanced with 50 samples for each species.
- Sepal Width contains a few natural outliers that were retained.
- The dataset is fully prepared for machine learning model development.
